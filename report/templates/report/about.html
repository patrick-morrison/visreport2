About Vis.Report

This site was built on Whadjuk Noongar boodja (country). The Noongar peoples have ongoing connections to this land and sea. I pay my respect to Elders past and present.

Vis.Report contains a database of visibility and condition reports for dive sites around Perth, all contributed by users. Clicking any site on the map will display the most recent report. Clicking on those will show a list of all reports for that site, and allow you to contribute your own (if you are signed in). Please share it around, the more people that contribute frequently, the more useful the information becomes! Dive sites are curated by me. If you have a suggestion for a site that isn't on here, please email vis.report.app@gmail.com.

The factors affecting dive conditions and visibility are not yet fully understood, particularly in the Swan River. I intend to use this data to better understand long-term visibility trends. Any publication of the data will acknowledge that it was contributed by citizen scientists via this platform. All data contributed is licensed under a Creative Commons Attribution-ShareAlike 4.0 International License. This means it is open for research or any other derivitive work, as long as it properly attributed and that work is also shared with the same conditions. JSON files for the sites ⬇️ and reports ⬇️ are dynamically generated straight from the database.

This site is built on the Django web framework. The code can be found on the Github page.

Safe diving!
